#  Pequenos Servos

Um aplicativo móvel simples e eficiente projetado para gerenciar e automatizar a escala de coroinhas em comunidades paroquiais.

##  Proposta do App
O objetivo principal é eliminar o trabalho manual e os conflitos na hora de montar as escalas de missas, equilibrando critérios importantes da comunidade:
* **Escalas Justas:** Sistema automático de pesos que prioriza quem serviu menos vezes para garantir oportunidades iguais.
* **Filtros Inteligentes:** Organização por nível de experiência (1 a 4), localidade, coral e disponibilidade (Sábado ou Domingo).
* **Armazenamento Local:** Funciona de forma 100% offline direto no celular do coordenador, sem a necessidade de um banco de dados externo complexo.

---

##  Tecnologias Utilizadas
O projeto foi construído utilizando as ferramentas mais modernas do ecossistema mobile atual:
* **React Native & Expo:** Framework para desenvolvimento de aplicativos nativos de alta performance.
* **Expo Router:** Navegação moderna baseada em arquivos (File-based routing).
* **AsyncStorage:** Banco de dados nativo do dispositivo para persistência dos dados de forma rápida e leve.
* **Expo Vector Icons (Ionicons):** Biblioteca de ícones utilizada para uma interface limpa e intuitiva.
